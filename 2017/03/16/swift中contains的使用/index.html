contains是swift中的一个实例方法，返回一个Bool值，用于做“是否包含”的判断。简单的说就是：告诉你一堆玩意儿中是否有你要的玩意儿。

方法contains()

用法1：判断字符串中是否包含某个字符或某段字符串

用法2：判断数组中是否包含某个元素

方法contains(where: )

该方法接收一个闭包，以序列元素作为闭包的参数，返回一个Bool值，判断该序列中是否有元素符合闭包条件。

用法1：判断Int数组中元素有没有大于100的
